# **鼠标连点器专业版(1.3.1)震撼来袭!**

## 简介
这是一个可以帮助用户自动完成重复的鼠标点击任务，提高工作效率，减少手动操作的疲劳。

## 功能和特点
- 自动连续点击鼠标左键。
- 可自定义点击间隔时间。
- 支持设置点击次数。
- 界面简洁，易于使用。

## 系统要求
- 操作系统：Windows 7/8/10/11
- 硬盘空间：至少152KB可用空间
